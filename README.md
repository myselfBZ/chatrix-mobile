# Chatrix Mobile

Mobile client for the Chatrix messaging system.

## Context
This app is part of the Chatrix system.

## Role in the system
- Native mobile client
- Connects to the Chatrix backend via HTTP/WebSocket
- Focused on real-time messaging.
